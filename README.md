Ansible Role - Node.js
======================

A node role to install node on elao symfony standard vagrant box

Requirements
------------

This role only run on elao symfony standard vagrant box. See https://vagrantcloud.com/elao/symfony-standard-debian

Example Playbook
----------------

    - hosts: servers
      roles:
         - { role: elao.node }
